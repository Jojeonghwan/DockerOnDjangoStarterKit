### How do I get set up? ###
* Docker 설치
* docker 명령어를 이용한 실행 
    * Production 모드로 이미지 생성 및 실행: $ docker-compose up -d --build 
    * Dev 모드로 이미지 생성 및 실행: $ docker-compose -f docker-compose-dev.yml up --build
